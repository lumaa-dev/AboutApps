About Cider Remote on Android

Cider Remote's development has changed multiple times across the year, before even Lumaa's arrival in the Cider Collective team. The current, up to date, Cider Remote app is a native iOS app, made in SwiftUI. For an "official" Cider Remote app, it would also need to be a native Android app.

Lumaa doesn't make native Android apps (yet), which means Cider Remote needs an Android developer contributor.
